# GitHub Lesson

## Basics
### Using a Pre-Existing Repository
---
    git clone [git-url]

### Updating Repository
---
    git pull

### Add, Commit, Push
---
    git add [file(s)]
    git commit -m 'your commit message'
    git push
*NOTE: This will push to whatever branch you're on.

## Branching
### Branch Creation
---
    git branch
    git [branchname]
    git checkout [branchname]

### Switching Branches
---
    git checkout [branchname]

### Merging Branches
---    
    git merge [branchname]
